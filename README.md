# 넓고 얕게 외워서 컴퓨터 전공자 흉내내기

## Intro

- 다나와에 가면

## 컴공 이론을 위한 기초체력 다지기

- 1비트와 디지털
- 4비트와 16진수 그리고 진법변환
- 16진수 표기가 사용되는 예
- 외워야 할 단위체계와 숫자
- 컴퓨터가 글자를 다루는 방법
- 컴퓨터가 사진을 다루는 방법

## 넓고 얕은 컴퓨터 구조

- 디지털 회로와 덧셈 (CPU 자작도 가능!)
- 컴퓨터가 뺄셈하는 방법
- CPU가 곱하고 나누는 방법
- 컴퓨터가 연산하는 과정
- 컴퓨터가 기억공간을 관리하는 방법
- HDD, SSD 그리고 파일시스템

## 넓고 얕은 운영체제

- 동시성과 병렬성
- 원자성, 동기화 그리고 교착상태
